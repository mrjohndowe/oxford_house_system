# Administrator Guide

## Overview

The Oxford Houses system allows administrators to manage houses, users, contracts, and reports from one central platform.

---

## User Management

Admins can:

- Create new users
- Assign roles
- Assign houses
- Activate / deactivate accounts

Access user management through:

Users & Access Manager

---

## House Management

Central admins can:

- Create houses
- Activate / deactivate houses
- Assign users to houses

---

## Contract Management

Contracts can be:

- Uploaded
- Reviewed
- Marked as Fulfilled
- Marked as Voided

Contract stamps require the contract password.

---

## Security Panel

Users can:

- Change their password

Admins can:

- Change contract password

Roles allowed to change contract password:

- House Manager
- Central Admin
- Super Admin

---

## Audit Logs

Audit logs track:

- User changes
- Permission changes
- System activity
- Contract updates
